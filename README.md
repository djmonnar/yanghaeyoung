# 양해영 경상남도의원 후보 홈페이지

국민의힘 기호 2 양해영 경상남도의원 후보 홍보 홈페이지입니다.

## 배포

GitHub Pages를 `main / root`로 설정하면 루트 `index.html`이 바로 홈페이지로 표시됩니다.

## 이미지 교체

실제 사진을 사용할 때는 아래 파일명으로 `images/` 폴더에 넣으면 됩니다.

- `images/yang-profile.jpg`
- `images/yang-meeting.jpg`
- `images/yang-speech.jpg`
- `images/yang-banner.jpg`
- `images/yang-character.jpg`

JPG 파일이 없을 때는 SVG fallback이 표시됩니다.
